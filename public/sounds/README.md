# Sonidos de Alarma

Esta carpeta contiene los archivos de audio para las notificaciones del sistema.

## Archivo requerido

- **alarma-cocina.mp3** (o .wav, .ogg): Sonido que se reproduce cuando llega un nuevo pedido a cocina.

## Recomendaciones

- **Formato**: MP3, WAV u OGG
- **Duración**: 0.5 - 1 segundo
- **Volumen**: Normalizado (no muy alto ni muy bajo)
- **Tipo**: Sonido de notificación/alerta claro y notorio

## Dónde conseguir sonidos

- Freesound.org
- Zapsplat.com
- SoundBible.com

Buscar: "notification sound", "alert beep", "kitchen bell"
